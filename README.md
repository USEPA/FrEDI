# FrEDI
The Framework for Evaluating Damages and Impacts (FrEDI), estimates the annual physical and economic impacts of temperature change to the US, through the 21st century (and optionally through 2300).

***

This code repository serves to maintain transparency of technical Agency work products by archiving previously published material that was subject to public and external peer review.
